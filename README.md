## 문제해결 방법과 순서

1. 팀원들 각자 코드 수정하고 commit 후 push하는 과정
2. 팀원들 다 푸쉬한다음 팀장이 메인에서 dev/a 병합하는데 충돌발생
2. 코드 충돌 해결 main과 dev/a
3. dev/b merge하고 dev/c하는데 충돌발생
4. 코드 수정 후 2번째 충돌 해결
5. 충돌 해결2한다음 log
6. 마지막으로 main에서 코드 실행시 발생하는 문제 수정 후 최종 커밋
7. 최종 실행 결과
